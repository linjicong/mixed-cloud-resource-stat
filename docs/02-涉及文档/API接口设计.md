# API接口设计文档

## 资源查询接口

### 华为云资源查询

#### 获取华为云资源列表

**接口地址：** `GET /huawei/{type}`

**接口描述：** 获取华为云指定类型的资源列表

**路径参数：**
- `type` (String, 必填): 资源类型，可选值：
  - `ecs`: 弹性云服务器
  - `rds`: 云数据库
  - `elb`: 负载均衡
  - `evs`: 云硬盘
  - `vpc`: 虚拟私有云
  - `eip`: 弹性公网IP
  - `ims`: 镜像服务
  - `cbr`: 云备份服务
  - `bills`: 月度成本账单

**响应示例：**
```json
[
  {
    "pk": 1,
    "statDate": "2024-01-01T00:00:00",
    "id": "xxx",
    "name": "xxx",
    ...
  }
]
```

**响应状态码：**
- `200`: 成功
- `400`: 不支持的资源类型
- `500`: 服务器内部错误

### 腾讯云资源查询

#### 获取腾讯云资源列表

**接口地址：** `GET /tencent/{type}`

**接口描述：** 获取腾讯云指定类型的资源列表

**路径参数：**
- `type` (String, 必填): 资源类型，可选值：
  - `cvm`: 云服务器
  - `cdb`: 云数据库
  - `clb`: 负载均衡
  - `cbs`: 云硬盘
  - `bills`: 账单资源汇总

#### 获取阿里云资源列表

**接口地址：** `GET /aliyun/{type}`

**接口描述：** 获取阿里云指定类型的资源列表

**路径参数：**
